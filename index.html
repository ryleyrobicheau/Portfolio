My Resort Dashboard

I designed and built a dashboard that displays current snow conditions for ski resorts in a specific region. The dashboard provides a centralized hub where users can quickly and easily view the latest snow conditions at multiple resorts and make informed decisions on where to ski that day. The dashboard uses the OpenWeather API to provide real-time updates on snow depth, temperature, and weather forecasts for each resort. Users can filter resorts by location and snow depth. The project aimed to provide a convenient and reliable tool for skiers to plan their day on the slopes. More details in the ReadMe on GitHub.
